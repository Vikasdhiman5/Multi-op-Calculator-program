## üìñ Usage Guide

After running the program:

1. The system will display a list of operations.
2. Enter your choice (1 to 6).
3. Enter two numbers.
4. The calculator will display the result.
5. Repeat the program as needed.

| Choice | Operation   |
|--------|-------------|

| 2      | Subtraction |
